### Rickroll

A program for MacOS and the TI 84 Plus CE that literally rickrolls you (I am Serious)

### Compiling for the TI 84 Plus CE
Install the CE toolchain, then Install the zip of this repo and run make.


### Compiling for MacOS
1. you need xcode Command Line Tools,install that via 
xcode-select --install
2. Download this repo
3. Then run clang++ basic.cpp -o RickRoll

### Windows and Linux
Use Any modern c++ Compiler to Compile Basic.cpp into an exe or a Linux app.

NOTE: on Windows,You MUST USE MSYS2 Because It Will save You A Bunch Of Time.
